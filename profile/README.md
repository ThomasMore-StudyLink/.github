## 🧠 What is StudyLink?

**StudyLink** is a desktop peer tutoring ecosystem built for university students. It connects students who need academic help with high-achieving peers who can tutor them — in a secure, accountable, and affordable way.

## 🚀 Our Project

## 🛡️ Key Features

- 🔍 **Smart Tutor Search** — Find tutors by course code or subject
- 📅 **Booking System** — Schedule and manage tutoring sessions
- 💳 **Payment Simulation** — Transparent cost calculation and balance tracking
- ⭐ **Review & Rating System** — Verified peer reviews for quality control
- 🛡️ **Admin Jury System** — Dispute resolution between students and tutors
